# Requirements: IntraChat

**Defined:** 2026-04-15
**Core Value:** Сотрудник открывает приложение и мгновенно видит живой корпоративный чат — сообщения приходят в реальном времени, файлы отправляются без лагов.

## v1 Requirements

### Authentication

- [ ] **AUTH-01**: Пользователь входит по email + пароль
- [ ] **AUTH-02**: Администратор создаёт аккаунты (не самостоятельная регистрация)
- [ ] **AUTH-03**: Сессия сохраняется после перезагрузки браузера
- [ ] **AUTH-04**: Деактивированный пользователь не может войти
- [ ] **AUTH-05**: Middleware перенаправляет неавторизованных на /login

### Database & Schema

- [x] **DB-01**: Таблица profiles (id, email, display_name, avatar_url, role, is_active)
- [x] **DB-02**: Таблицы groups + group_members (иерархия, parent_id)
- [x] **DB-03**: Таблицы folders + folder_items (личные папки)
- [x] **DB-04**: Таблица messages (soft delete, reply_to, attachments FK)
- [x] **DB-05**: Таблица message_reads (статусы прочтения)
- [x] **DB-06**: Таблица message_reactions (эмодзи реакции)
- [x] **DB-07**: Таблицы direct_chats + direct_messages
- [x] **DB-08**: RLS включён для всех таблиц
- [ ] **DB-09**: Realtime включён для messages и direct_messages
- [ ] **DB-10**: Storage bucket "attachments" настроен с политиками

### Groups & Navigation

- [ ] **GRP-01**: Левая панель показывает список групп с иерархией (2 уровня)
- [ ] **GRP-02**: Личные папки отображаются в sidebar
- [ ] **GRP-03**: Пользователь может перемещать группы в папки
- [ ] **GRP-04**: Активная группа подсвечена в sidebar
- [ ] **GRP-05**: Название группы и кол-во участников в шапке чата

### Messaging

- [ ] **MSG-01**: Сообщения доставляются в реальном времени через Realtime
- [ ] **MSG-02**: Пользователь видит историю сообщений (pagination/infinite scroll)
- [ ] **MSG-03**: Reply на сообщение (цитата с именем и текстом)
- [ ] **MSG-04**: Сообщение помечается как прочитанное при открытии чата
- [ ] **MSG-05**: Статус ✓ (доставлено) / ✓✓ (прочитано) для своих сообщений
- [ ] **MSG-06**: Индикатор "печатает..." (Supabase Presence)
- [ ] **MSG-07**: Пользователь может удалить своё сообщение (soft delete)
- [ ] **MSG-08**: Реакции (эмодзи) на сообщение — добавить/убрать

### Attachments

- [ ] **ATT-01**: Отправка изображений (JPEG, PNG, GIF, WebP, ≤10MB)
- [ ] **ATT-02**: Отправка файлов (≤50MB, любой тип)
- [ ] **ATT-03**: Голосовые сообщения (запись через браузер, WebM, ≤5MB)
- [ ] **ATT-04**: Видео-кружки (запись через камеру, MP4, ≤30MB, ≤60 сек)
- [ ] **ATT-05**: Превью изображений в чате (клик — полный просмотр)
- [ ] **ATT-06**: Видео-кружки отображаются круглыми с воспроизведением
- [ ] **ATT-07**: Файлы показываются как карточка с именем и размером
- [ ] **ATT-08**: Signed URLs для скачивания (не публичные ссылки)

### Direct Messages

- [ ] **DM-01**: Пользователь может начать DM с любым сотрудником
- [ ] **DM-02**: DM показываются в sidebar отдельным разделом
- [ ] **DM-03**: Все функции сообщений работают в DM (вложения, реакции, reply)
- [ ] **DM-04**: Статус прочтения в DM

### Search

- [ ] **SCH-01**: Поиск по названиям групп и чатов
- [ ] **SCH-02**: Полнотекстовый поиск по сообщениям (в открытом чате)

### Pinned Messages

- [ ] **PIN-01**: Администратор/участник может закрепить сообщение
- [ ] **PIN-02**: Закреплённое сообщение отображается в шапке чата

### Admin Panel

- [ ] **ADM-01**: Список всех пользователей с возможностью создать/деактивировать
- [ ] **ADM-02**: Создание/редактирование/удаление групп
- [ ] **ADM-03**: Управление участниками групп
- [ ] **ADM-04**: Доступ только для роли admin (проверка на сервере)

### PWA

- [ ] **PWA-01**: Web App Manifest (имя, иконки, цвет темы)
- [ ] **PWA-02**: Service Worker для кэширования (next-pwa)
- [ ] **PWA-03**: Кнопка "Добавить на главный экран" работает на iOS и Android
- [ ] **PWA-04**: Метатеги для корректного отображения при установке

## v2 Requirements

### Notifications

- **NOTF-01**: Push-уведомления при новых сообщениях
- **NOTF-02**: Бейдж непрочитанных на иконке группы

### Advanced Features

- **ADV-01**: Пересылка сообщений
- **ADV-02**: Редактирование отправленного сообщения
- **ADV-03**: Threads/ветки обсуждений
- **ADV-04**: Экспорт истории чата

## Out of Scope

| Feature | Reason |
|---------|--------|
| Внешний OAuth (Google, Apple) | Корпоративный продукт, email+password достаточно |
| Видеозвонки | Отдельная задача, высокая сложность инфраструктуры |
| Боты и интеграции | v2+ |
| E2E-шифрование | Не является корпоративным требованием |
| iOS/Android нативные приложения | PWA достаточно для ~100 пользователей |
| Самостоятельная регистрация | Только через администратора |
